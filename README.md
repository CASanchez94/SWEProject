# webApp_spring2026_04

Xavier Amico, Fabian Garza, Chris Sanchez


Description

This project is a web application built using the Django framework, designed to enhance campus life by fostering academic collaboration among students within the same major. 
It connects lowerclassmen with upperclassmen who have already completed relevant courses, creating opportunities for a deeper understanding of course material. The platform is especially useful for students who cannot attend TA sessions or prefer group-based learning environments.

Features

Shared scheduling/calendar system to coordinate study sessions

Goal-setting or agenda builder to keep meetings structured and productive

Chat Feature and Profile Pages.

Goal

The goal of this project is to promote stronger peer connections, improve academic performance, and create a more wholesome college experience.
