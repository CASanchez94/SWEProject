# UTRGV Study Group Web Application
Group: Xavier Amico, Fabian Garza, Chris Sanchez

## Project Description

This web application is designed to bring students together, strengthen campus connections, and support academic success by enabling the creation of study groups within the same major. Built using the Django framework, the platform helps students collaborate more effectively outside the classroom.

While many students may not always attend TA sessions or may prefer collaborative learning environments, this application provides an alternative by connecting lowerclassmen with upperclassmen who have previously taken the same courses and can offer guidance based on their experience.

## Key Features

The application will include several tools to support productive and organized study sessions:

Study Group Matching: Allows students within the same major to discover and join study groups.

Shared Scheduling System: A calendar feature that helps members coordinate meeting times.

Agenda / Goal Builder: Enables groups to create structured plans for each study session.

File Sharing: Allows users to upload and distribute study materials such as notes, practice problems, and resources.

## Project Goal

The goal of this project is to create a stronger sense of community at UTRGV while helping students succeed academically through peer collaboration. The platform promotes meaningful connections and more productive group learning experiences by offering tools for communication, organizing, and resource sharing.
